Le icone PWA devono essere create manualmente.

Puoi usare uno strumento online come:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
- https://favicon.io/

Oppure creare manualmente due file PNG:
- icon-192.png (192x192 pixel)
- icon-512.png (512x512 pixel)

Per ora il file icon.svg nella cartella può essere usato come riferimento per il design.
Il design mostra un orologio stilizzato con sfondo blu scuro (#2c3e50) e cerchio blu (#3498db).
